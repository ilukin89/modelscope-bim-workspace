# Product

## Register

product

## Users

BIM coordinators, architects, engineers, and construction project leads reviewing dense federated models on desktop displays. They need to orient themselves quickly, isolate model systems, inspect objects, and resolve coordination issues without losing viewport context.

## Product Purpose

ModelScope is a focused code-first prototype for a professional 3D model review workspace. It demonstrates credible interaction patterns around model navigation, object inspection, review issues, and AI-assisted analysis without attempting to implement a real BIM engine.

## Brand Personality

Calm, precise, and technically confident. The interface should feel familiar to users of professional design and engineering tools, with density that supports expertise rather than decoration.

## Anti-references

Generic analytics dashboards, marketing-style SaaS cards, purple gradients, glassmorphism, decorative illustrations, oversized rounded containers, and interfaces that treat a technical viewport as secondary content.

## Design Principles

1. Keep the viewport dominant and preserve spatial context.
2. Make system state legible through consistent semantic tokens.
3. Favor familiar professional-tool patterns over novel navigation.
4. Support dense information with clear hierarchy and compact rhythm.
5. Use AI as a review aid with explicit, inspectable suggestions.

## Accessibility & Inclusion

Target WCAG 2.1 AA contrast, visible keyboard focus, text labels for icon controls, non-color-only state indicators, reduced reliance on motion, and layouts that avoid horizontal overflow at narrower viewport sizes.
